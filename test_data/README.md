# Test Data Directory

This directory contains Excel files with test data for API testing.

## Structure:

- **JSON/**: Excel files with JSON payload test data
- **XML/**: Excel files with XML payload test data

## Excel Format:

### For JSON Tests:
The Excel file should have columns like:
- title
- description
- release_year
- language_id
- rental_duration
- rental_rate
- length
- replacement_cost
- rating
- special_features
- expected_response

### For XML Tests:
Similar structure with XML-specific fields.
